# SSPCloud LLM

Small public repository containing the deployment contracts for the LLMs available on the SSPCloud (https://llm.lab.sspcloud.fr).
This repository is continuously deployed via ArgoCD.  
Contributions (bug fixes, optimizations, etc.) are welcome via Pull Request. However, please note that because resources are limited, requests for new models may not necessarily be accepted.  
Status of deployed models: https://status.lab.sspcloud.fr/status/sspcloud
